SpringBoot

学习Springboot 主要为了完成自己对微服务的实践。
介绍什么的不再赘述。

框架、组件(比如SpringCloud包含的组件写到的时候再补充)

  • SpringBoot
  • SpringCloud
  • Redis、MongoDB、mysql
  • Hystrix
  • Kafka、RebittMQ
  • Shiro、Spring Security
  • Oauth2认证方式
  • thymeleaf
  • gradle、Maven
  • git

服务

  • common：用于公共部分
  • discovery: 注册中心
  • configserver: 配置中心
  • apiGateway: 服务网关
  • Ribbon：负载均衡（基于HTTP与TCP）
  • Feign: 声明式web service客户端
  • Oauth2 : 认证鉴权
